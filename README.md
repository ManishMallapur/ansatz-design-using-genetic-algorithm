## ğŸ“– Overview

This repository provides the quantum circuits evolved using a Genetic Algorithm (GA) to maximize expressibility under fixed qubit and depth constraints.

The repository includes:

- Evolved circuit architectures (JSON format)
- Expressibility evolution logs (TXT)
- Visualization plots (PNG)
- Implementation notebooks

The circuits are designed to be directly reusable in VQE and other variational quantum algorithms.

**Fields:**

- `generation` â†’ GA generation number  
- `qubits` â†’ number of qubits  
- `layers` â†’ circuit depth  
- `expressibility` â†’ Jensenâ€“Shannon divergence score (**lower = better**)

## ğŸ“ˆ Expressibility Metric

Expressibility is computed as the Jensenâ€“Shannon divergence (JSD) between:

- The fidelity distribution generated by the parameterized ansatz  
- The theoretical Haar-random fidelity distribution  

Lower JSD â†’ Circuit distribution closer to Haar â†’ Higher expressibility  

The Genetic Algorithm minimizes this metric during evolution.
